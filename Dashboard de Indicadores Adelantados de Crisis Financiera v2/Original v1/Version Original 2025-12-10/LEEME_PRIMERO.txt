🚀 INICIO RÁPIDO (5 MINUTOS):
═══════════════════════════

1. Instalar Python 3.8+ (si no lo tienes)
   → https://www.python.org/downloads/

2. Instalar librerías:
   pip install pandas numpy yfinance fredapi requests beautifulsoup4 openpyxl python-dotenv

3. Obtener API Key GRATIS de FRED:
   → https://fred.stlouisfed.org/docs/api/api_key.html

4. Crear archivo .env con:
   FRED_API_KEY=tu_clave_aqui

5. Ejecutar:
   python crisis_dashboard_pro.py

6. Esperar 2-3 minutos (primera vez con backtesting)

7. ¡LISTO! Verás resultados en pantalla + Excel generado

🆘 PROBLEMAS COMUNES:
══════════════════════

P: Error "No se encontró FRED_API_KEY"
R: Crea archivo .env con tu clave

P: Error "Rate Limit" de Yahoo
R: El sistema reinicia automáticamente, espera 30s

P: VIX no descarga
R: El sistema usa FRED automáticamente como backup

P: Excel no se genera
R: Instala: pip install openpyxl
